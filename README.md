# gif maker (python)

a desktop application built with python that converts video files into GIFs with live preview, adjustable FPS, and scaling controls.

## features
- select any video file
- live preview inside the app
- adjustable frame rate (fps)
- adjustable output scale
- one-click gif export
- progress bar during rendering

## tech stak
- python
- opencv
- tkinter
- pillow (pil)
- threading

## installation

pip install opencv-python pillow

## run the app

python gif_maker.py

## why?
the following demonstrates gui development, real-time video processing, and multithreaded media exporting.
